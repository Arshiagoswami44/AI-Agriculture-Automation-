# AI-Agriculture-Automation-

AI-Powered Agriculture Automation
An intelligent system designed to monitor agricultural conditions and provide predictive insights for crop optimization. This project integrates IoT simulation, Machine Learning, and Database Management to automate agricultural decision-making.

🚀 Features
Crop Prediction: Uses a Random Forest Classifier to recommend the best crop based on soil and weather parameters (N, P, K, Temperature, Humidity, pH, Rainfall).

Real-time Simulation: Simulates sensor data input for testing automation logic.

Database Integration: Automatically logs sensor readings and AI predictions into a MySQL database for historical tracking.

Alert System: (Optional) Console-based alerts for abnormal environmental conditions (e.g., extreme low humidity).

🛠️ Tech Stack
Language: Python 3.x

Machine Learning: Scikit-learn, Pandas, NumPy

Database: MySQL

Tools: Jupyter Notebook (for model training), VS Code
